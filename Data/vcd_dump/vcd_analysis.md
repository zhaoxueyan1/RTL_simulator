# VCD Analysis Format

## 1 Analysis file

suffix : ```.analysis```

### 1.1 Content

```Version```, ```Comment```, ```Date```, ```Signal count```,```Times Recorded```

### 1.2 Sample

```
Version:        Generated by VerilatedVcd 
Comment:       
Date:           Wed Jul  8 19:34:09 2020
 
Signal count:  8065
Times Recorded:1231
```

## 2 Output file

suffix: ```.out```
### 2.1 Read method

```C++
while(~scanf("%d %s %d",&num,signal_name,&signal_id)){
    scanf("%d %s",&time_step,val);
}
```
